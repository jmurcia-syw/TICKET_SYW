# 03_Deliveries

Aquí vive una carpeta por cada Delivery: `DEL-XXX/`.

Una Delivery es un corte de versión de la app completa, aceptado formalmente tras acumular las iteraciones necesarias (ver `../README.md`, sección "Modelo de trabajo").

Para generar una nueva Delivery:
1. Crea `DEL-XXX/` (siguiente consecutivo).
2. Copia `../00_Templates/DELIVERY_TEMPLATE.md` como `DEL-XXX/DEL-XXX.md`.
3. Completa el acta de aceptación al cierre.
