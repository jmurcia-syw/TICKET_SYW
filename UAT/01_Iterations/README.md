# 01_Iterations

Aquí vive una carpeta por cada iteración de pruebas: `ITER-XXX/`.

Para iniciar una nueva iteración:
1. Crea `ITER-XXX/` (siguiente consecutivo).
2. Copia `../00_Templates/ITERATION_TEMPLATE.md` como `ITER-XXX/ITER-XXX.md`.
3. Crea `ITER-XXX/images/` y `ITER-XXX/attachments/` según necesites.
