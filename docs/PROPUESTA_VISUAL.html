Calendario Laboral — Configuración SLA

Define los horarios hábiles, festivos y excepciones. El motor de SLA solo cuenta tiempo dentro de estas franjas.

Cómo usa el SLA este calendario
Un ticket P2 tiene 8 horas hábiles de resolución. Si llega el viernes a las 17:00 y el equipo trabaja hasta las 18:00, solo cuenta 1 hora ese día. El reloj se pausa a las 18:00 y reanuda el lunes a las 08:00. Los sábados, domingos y festivos no se descuentan del SLA.

Cómo lo consume el motor de SLA

El calendario se guarda en PostgreSQL. El SLAService lo lee al crear cada ticket y calcula la fecha de vencimiento adelantando solo dentro de las franjas hábiles.

Tablas en PostgreSQL

  start_time,        -- 08:00

SLAService — lógica central
def calculate_due_date(
  start: datetime,
  hours_sla: float
) -> datetime:

  remaining = hours_sla
  cursor = start

  while remaining > 0:
    if is_working_moment(cursor):
      slot_end = next_slot_end(cursor)
      slot_h = hours_until(cursor, slot_end)
      if slot_h >= remaining:
        return cursor + timedelta(
               hours=remaining)
      remaining -= slot_h
      cursor = slot_end
    else:
      cursor = next_slot_start(cursor)
Ejemplo real: Ticket P2 (8h) creado el viernes a las 17:00 → 1h el viernes (17:00–18:00) + 7h el lunes (08:00–15:00) = vence el lunes a las 15:00. Sábado y domingo no se cuentan.

is_resolver_available() — lógica
def is_resolver_available(rid, at):

  # 1. Hora hábil de empresa?
  if not sla.is_working_moment(at):
    return False

  # 2. Tiene ausencia ese día?
  absence = Absences.get(rid, at.date())
  if absence:
    if absence.type in (
      'vacation','sick','training'):
      return False
    if absence.type == 'partial':
      if (absence.partial_from
          <= at.time()
          <= absence.partial_until):
        return False

  # 3. Horario personal diferente?
  personal = Schedule.get(rid, at.weekday())
  if personal:
    return (personal.is_active and
      personal.start <= at.time()
               < personal.end)

  return True  # usa empresa por defecto
